# BookStorage

A Java project which adds and stores and helps search/compare books with their price, sales, country, author, and more

## Features

- Add/Store Multiple Books
- Get Sales Info
- Get Price Info
- Get Book Details
- Modify some elements (Author and Publisher)

## Software Used

- Java

## Requirements

- Java JDK 21+ from testing

## Instructions

- When starting the project, there will be a main menu displaying all the features mentionned above with their own number.
- After choosing one of the features by typing their corresponding number, there will be many prompts to answer in order to add the book or modify an element of the book.
- The get features will only display the result.
